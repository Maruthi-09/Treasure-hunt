# Treasure Hunt Game

A text-based adventure game built with Python where players make decisions to progress through different paths in search of hidden treasure.

## Description

This interactive game presents players with a series of choices. Each decision determines the outcome of the adventure, leading either to victory or a game over. The objective is to make the correct choices and successfully find the treasure.

## Features

- Interactive text-based gameplay
- Multiple decision paths
- Different game endings
- Simple command-line interface
- Beginner-friendly Python project

## Technologies Used

- Python

## How to Run

1. Download or clone this repository.
2. Open the project in PyCharm or any Python IDE.
3. Run `treasure-hunt-game.py`.
4. Follow the on-screen prompts and make your choices.

## Objective

Navigate through the adventure by making the correct decisions and discover the hidden treasure.
